API de Elementos


A API de Elementos permite o acesso aos dados de elementos químicos armazenados no banco de dados da aplicação, retornando informações essenciais como símbolo, massa atômica, número de prótons, entre outros detalhes. Abaixo estão descritos os endpoints disponíveis para listar todos os elementos e buscar informações específicas sobre um elemento individual.

• Obs: nesta API, o único método HTTP disponível é o GET, sendo usado apenas para leitura/visualização dos dados.

Função Endpoint Método
Retorna uma lista de elementos <base_url>/api/elementos/ GET
Retorna um elemento específico <base_url>/api/elementos/<elemento>/ GET

Lista de Elementos


HTTP Method: GET
Endpoint: /api/elementos/

O banco de dados possui variadas informações sobre os elementos químicos, e este endpoint serve para retornar uma lista com os elementos informações de forma simplificada.

Dados retornados
Campo Descrição Tipo
atomic_number Número atômico do elemento int
name Nome do elemento/ string
symbol Símbolo químico do elemento string
atomic_mass Massa atômica do elemento float
state_matter Estado físico do elemento à temperatura ambiente string
eletronic_configuration Configuração eletrônica do elemento string

Consultar Elemento Específico


HTTP Method: GET
Endpoint: /api/elementos/<elemento>/

Para pesquisar um elemento específico, basta adicionar ao fim da URL "api/elementos/", o símbolo do elemento que seja pesquisar. Exemplo: /api/elementos/Fe/ -> Deste modo, você estará pesquisando pelo elemento Ferro.

Diferente da lista, esta pesquisa retorna dados a mais sobre o elemento.

Dados retornados
Campo Descrição Tipo
atomic_number Número atômico do elemento int
name Nome do elemento/ string
symbol Símbolo químico do elemento string
atomic_mass Massa atômica do elemento float | string
state_matter Estado físico do elemento à temperatura ambiente string
eletronic_configuration Configuração eletrônica string
electron_distribution Distribuição de elétrons string
electrons_qty Quantidade de elétrons int
neutrons_qty Quantidade de nêutrons int
ionic_states Estados iônicos string
density Densidade float | string
melting_point Ponto de fusão float
boiling_point Ponto de ebulição float
additional_information Informações adicionais do elemento string
