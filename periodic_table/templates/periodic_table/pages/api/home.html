Documentação da API do Site


Bem-vindo à documentação oficial da API do Projeto Tabela Periódica. Esta API fornece acesso a dados relacionados a elementos químicos, atividades educacionais, autenticação de usuários, e mais. Desenvolvida com o Django REST Framework, a API foi projetada para ser segura e fácil de usar, permitindo que desenvolvedores integrem recursos do Projeto Tabela Periódica em suas próprias aplicações.


Os endpoints da API seguem um padrão simples e organizado. Cada app possui uma URL inicial específica seguida por /api/, exceto pelo app periodic_table, que utiliza diretamente /api/. Os caminhos ficam da seguinte forma:

App URL inicial Dados / Tabelas trabalhadas
periodic_table /api/.../ Elements
users /usuarios/api/.../ Tokens, Users
learn_lab /learn-lab/api/.../ Activities, Ratings

Recursos Disponíveis


  • Consulta e detalhamento de elementos químicos: obtenha propriedades detalhadas de cada elemento.
  • Gerenciamento de usuários: incluindo criação, consulta e atualização de perfis.
  • Gestão de atividades educativas: crie, liste, edite, e exclua atividades de aprendizado.
  • Autenticação segura com JWT: controle de acesso através de Tokens de Segurança.


Pré-requisitos


  • Django e Django REST Framework instalados na sua aplicação.
  • Bibliotecas de Autenticação JWT: configuradas para gerenciar a autenticação de tokens.
  • Um ambiente de desenvolvimento configurado, idealmente com um ambiente virtual Python.


Instalação e Configuração


  • Clone o repositório e instale as dependências necessárias.
  • Configure as variáveis de ambiente necessárias, como as chaves de API e secret keys, caso sejam requisitadas.
  • Certifique-se de ter o Django configurado e a API ativa para começar a utilizá-la.

Consulte a documentação do projeto para mais informações sobre instalação e configuração: github.com/Ric002x/TabelaPeriodica
